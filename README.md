## MP3Tag-Docker-VNC

Docker Version of MP3Tag (Wine on Ubuntu Jammy)

Based on [linuxserver's docker-baseimage-kasmvnc](https://github.com/linuxserver/docker-baseimage-kasmvnc)



 ## Options
| Host  | Container|
| ------------- | ------------- |
| Folder containing audio files  | /data |
| HTTP Port  | 3000  |
| HTTPS Port  | 3001  |
